# Landing Page Project

This project is a simple landing page designed to provide information and allow users to subscribe via a Google Form. 

## Project Structure

```
landing-page/
├── index.html       # Main landing page
├── style.css        # Styles for the landing page
├── script.js        # JavaScript for interactivity
└── README.md        # Project documentation
```

## Files Overview

- **index.html**: Contains the main structure of the landing page, including a header, sections for class schedules, and a subscription form.
- **style.css**: Provides styling for the landing page to ensure a clean and responsive design.
- **script.js**: Contains JavaScript code for any interactivity on the page.
- **README.md**: This file provides an overview of the project and instructions for use.

## Instructions

1. Clone the repository or download the files.
2. Open `index.html` in a web browser to view the landing page.
3. Customize the Google Form link in `index.html` to your specific form URL.
4. Modify `style.css` to change the appearance as needed.
5. Use `script.js` for any additional JavaScript functionality you may want to implement.
